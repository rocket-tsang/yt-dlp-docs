# 개인정보처리방침

최근 업데이트 날짜: 2026년 3월 26일

## 1. 서론

yt-dlp 기술 문서 웹사이트에 오신 것을 환영합니다. 본 사이트는 사용자의 개인정보를 소중하게 생각하며, 이를 보호하기 위해 최선을 다하고 있습니다. 본 방침은 정보 수집, 이용 및 보호 방식에 대해 설명합니다.

## 2. 정보 수집

### 2.1 자동 수집 정보

본 웹사이트 방문 시 다음과 같은 정보가 자동으로 수집될 수 있습니다:

- **IP 주소**: 보안 방어 및 트래픽 분석용
- **브라우저 종류**: 웹사이트 디스플레이 최적화용
- **방문 시간**: 통계 분석용
- **방문 페이지**: 사용자 니즈 파악용

### 2.2 수집하지 않는 정보

- 개인 식별 정보 (성명, 이메일, 전화번호 등)
- 민감한 개인정보
- 쿠키 (본 사이트는 추적용 쿠키를 사용하지 않습니다)

## 3. 정보 이용

수집된 정보는 다음 목적으로만 사용됩니다:

- 웹사이트 콘텐츠 개선 및 사용자 경험 향상
- 트래픽 통계 분석
- 웹사이트의 안전한 운영 보장
- 법령 및 규정 준수

## 4. 정보 보호

### 4.1 기술적 조치

다음 보안 조치를 통해 정보를 보호하고 있습니다:

- **HTTPS 암호화**: 모든 데이터 전송에 SSL/TLS 암호화 적용
- **접근 제어**: 서버 및 데이터에 대한 접근 제한
- **보안 모니터링**: 웹사이트 보안 상태 실시간 감시

### 4.2 데이터 저장

- 본 웹사이트는 GitHub Pages를 통해 호스팅됩니다.
- 로그 데이터는 보안 서버에 저장됩니다.
- 사용자의 데이터를 판매하거나 공유하지 않습니다.

## 5. 제3자 서비스

본 웹사이트는 다음과 같은 제3자 서비스를 이용할 수 있습니다:

### 5.1 GitHub Pages

- **서비스 제공자**: GitHub, Inc.
- **용도**: 웹사이트 호스팅 서비스
- **개인정보처리방침**: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

### 5.2 외부 링크

본 사이트는 타 사이트로의 링크를 포함할 수 있습니다. 해당 사이트의 개인정보 보호 관행에 대해서는 책임을 지지 않으므로, 각 사이트의 방침을 확인하시기 바랍니다.

## 6. 사용자의 권리

사용자는 다음과 같은 권리가 있습니다:

- 수집된 본인 정보에 대한 열람 권리
- 정보의 정정 또는 삭제 요청 권리
- 특정 정보 수집 및 이용 거부 권리
- 개인정보 관련 문의 또는 민원 제기 권리

## 7. 쿠키 정책

### 7.1 쿠키 사용

본 웹사이트는 **추적용 쿠키를 사용하지 않습니다**.

### 7.2 필수 기술 쿠키

기능적 쿠키가 다음을 위해 사용될 수 있습니다:
- 웹사이트의 정상적인 작동 유지
- 사용자 설정 저장 (예: 언어 선택)

## 8. 아동 개인정보 보호

본 웹사이트는 13세 미만 아동을 대상으로 하지 않습니다. 아동의 개인정보를 의도적으로 수집하지 않으며, 실수로 수집된 것을 인지할 경우 즉시 연락해 주시기 바랍니다.

## 9. 방침 변경

본 개인정보처리방침은 수시로 업데이트될 수 있습니다. 변경 시 본 페이지에 공지하고 "최근 업데이트 날짜"를 갱신합니다. 정기적으로 확인해 보시기 바랍니다.
