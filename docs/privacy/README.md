# 隐私权政策

最后更新日期：2026年3月26日

## 1. 引言

欢迎访问 yt-dlp 技术文档网站。我们重视您的隐私，并致力于保护您的个人信息。本隐私权政策说明了我们如何收集、使用和保护您的信息。

## 2. 信息收集

### 2.1 自动收集的信息

当您访问本网站时，我们可能会自动收集以下信息：

- **IP 地址**：用于安全防护和流量分析
- **浏览器类型**：用于优化网站显示效果
- **访问时间**：用于统计分析
- **访问页面**：用于了解用户需求

### 2.2 我们不收集的信息

- 个人身份信息（姓名、邮箱、电话等）
- 敏感个人信息
- Cookie（本网站不使用跟踪 Cookie）

## 3. 信息使用

我们收集的信息仅用于：

- 改善网站内容和用户体验
- 进行流量统计分析
- 确保网站安全运行
- 遵守法律法规要求

## 4. 信息保护

### 4.1 技术措施

我们采取以下安全措施保护您的信息：

- **HTTPS 加密**：所有数据传输均使用 SSL/TLS 加密
- **访问控制**：限制对服务器和数据的访问
- **安全监控**：实时监控网站安全状态

### 4.2 数据存储

- 本网站托管于 GitHub Pages
- 日志数据存储于安全的服务器
- 我们不会出售或共享您的数据

## 5. 第三方服务

本网站可能使用以下第三方服务：

### 5.1 GitHub Pages

- **服务提供商**：GitHub, Inc.
- **用途**：网站托管服务
- **隐私政策**：https://docs.github.com/zh/site-policy/privacy-policies/github-privacy-statement

### 5.2 外部链接

本网站包含指向其他网站的链接。我们不对这些网站的隐私做法负责。建议您阅读每个网站的隐私政策。

## 6. 您的权利

您有权：

- 了解我们收集了哪些关于您的信息
- 要求更正或删除您的信息
- 拒绝某些信息的收集和使用
- 提出隐私相关的问题或投诉

## 7. Cookie 政策

### 7.1 Cookie 使用

本网站**不使用跟踪 Cookie**。

### 7.2 必要的技术 Cookie

某些功能性 Cookie 可能用于：
- 保持网站正常运行
- 记住您的偏好设置（如语言选择）

## 8. 儿童隐私

本网站不针对 13 岁以下的儿童。我们不会故意收集儿童的个人信息。如果您发现我们无意中收集了儿童的信息，请联系我们。

## 9. 政策更新

我们可能会不时更新本隐私权政策。更新后，我们会在本页面发布新版本并更新"最后更新日期"。建议您定期查看本政策。

## 10. 联系我们

如果您对本隐私权政策有任何疑问或建议，请通过以下方式联系我们：

- **GitHub Issues**：[提交问题](https://github.com/rocket-tsang/yt-dlp-docs/issues)
- **网站**：[https://github.com/rocket-tsang/yt-dlp-docs](https://github.com/rocket-tsang/yt-dlp-docs)

## 11. 适用法律

本隐私权政策受中华人民共和国法律管辖。如有争议，双方应友好协商解决。
