# プライバシーポリシー

最終更新日：2026年3月26日

## 1. はじめに

yt-dlp 技術ドキュメントサイトへようこそ。私たちはユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーでは、情報の収集、使用、および保護の方法について説明します。

## 2. 情報の収集

### 2.1 自動的に収集される情報

本サイトにアクセスすると、以下の情報が自動的に収集される場合があります：

- **IP アドレス**: セキュリティ保護とトラフィック分析に使用されます。
- **ブラウザの種類**: サイトの表示を最適化するために使用されます。
- **アクセス時間**: 統計分析に使用されます。
- **閲覧ページ**: ユーザーのニーズを把握するために使用されます。

### 2.2 収集しない情報

- 個人を特定できる情報（氏名、メールアドレス、電話番号など）
- 敏感な個人情報
- クッキー（本サイトは追跡用クッキーを使用しません）

## 3. 情報の使用

収集した情報は以下の目的のみに使用されます：

- サイトのコンテンツとユーザー体験の向上
- トラフィックの統計分析
- サイトの安全な運用の確保
- 法規制の遵守

## 4. 情報の保護

### 4.1 技術的措置

以下のセキュリティ対策を講じて情報を保護しています：

- **HTTPS 暗号化**: すべてのデータ転送は SSL/TLS で暗号化されています。
- **アクセス制御**: サーバーおよびデータへのアクセスを制限しています。
- **セキュリティ監視**: サイトのセキュリティ状態をリアルタイムで監視しています。

### 4.2 データの保存

- 本サイトは GitHub Pages でホストされています。
- ログデータは安全なサーバーに保存されます。
- データを販売したり、共有したりすることはありません。

## 5. サードパーティサービス

本サイトでは、以下のサードパーティサービスを使用する場合があります：

### 5.1 GitHub Pages

- **サービス提供者**: GitHub, Inc.
- **用途**: ウェブサイトのホスティングサービス
- **プライバシーポリシー**: https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement

### 5.2 外部リンク

本サイトには他サイトへのリンクが含まれています。それらのサイトのプライバシー慣行については責任を負いません。各サイトのプライバシーポリシーを読むことをお勧めします。

## 6. ユーザーの権利

ユーザーには以下の権利があります：

- 収集された情報の内容を知る権利
- 情報の訂正または削除を求める権利
- 特定の情報の収集と使用を拒否する権利
- プライバシーに関する質問や苦情を申し立てる権利

## 7. クッキーポリシー

### 7.1 クッキーの使用

本サイトは**追跡用クッキーを使用しません**。

### 7.2 必須の技術的クッキー

特定の機能的クッキーが以下のために使用される場合があります：
- サイトを正常に動作させるため
- ユーザーの優先設定（言語選択など）を記憶するため

## 8. 子供のプライバシー

本サイトは 13 歳未満の子供を対象としていません。子供から意図的に個人情報を収集することはありません。誤って子供の情報を収集したことに気付いた場合は、ご連絡ください。

## 9. ポリシーの更新

本プライバシーポリシーは随時更新されることがあります。更新された場合は、このページに新しいバージョンを掲載し、「最終更新日」を更新します。定期的に本ポリシーを確認することをお勧めします。

## 10. お問い合わせ

本プライバシーポリシーに関するご質問やご提案がございましたら、以下の方法でご連絡ください：

- **GitHub Issues**: [問題を報告](https://github.com/rocket-tsang/yt-dlp-docs/issues)
- **ウェブサイト**: [https://github.com/rocket-tsang/yt-dlp-docs](https://github.com/rocket-tsang/yt-dlp-docs)

## 11. 準拠法

本プライバシーポリシーは中華人民共和国の法律に準拠します。紛争が生じた場合は、双方が友好的な協議により解決するものとします。
