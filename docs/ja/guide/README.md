# yt-dlp プロジェクト紹介

## プロジェクトの概要

yt-dlp は強力なコマンドラインビデオダウンロードツールであり、youtube-dl の拡張版フォークです。このプロジェクトはコミュニティによって維持されており、より高速なダウンロード速度、より多くのサイトへの対応、そして豊富な機能を提供することを目的としています。

### プロジェクトの背景

youtube-dl は歴史あるビデオダウンロードツールですが、メンテナンスが停滞しているため、コミュニティは以下の点に重点を置いた yt-dlp フォークを作成しました：

- 🚀 より迅速な開発サイクル
- 🔧 より多くの機能と特性
- 🌐 より広範なサイトのサポート
- 🐛 より迅速なバグ修正

### プロジェクト情報

- **GitHub**: [https://github.com/yt-dlp/yt-dlp](https://github.com/yt-dlp/yt-dlp)
- **ライセンス**: Unlicense License
- **言語**: Python
- **プラットフォーム**: Windows, Linux, macOS

## 主要な機能

### 1. 広範なサイトサポート

yt-dlp は **1000以上** のビデオサイトをサポートしています。以下を含みますが、これらに限定されません：

- YouTube
- Bilibili (ニコニコ動画)
- Twitter/X
- Instagram
- Facebook
- TikTok
- Vimeo
- その他多数...

### 2. 強力なダウンロード機能

#### ビデオダウンロード
- 複数のビデオ形式と解像度をサポート
- 最適な品質を自動選択
- プレイリスト全体のダウンロードをサポート
- ライブストリームのダウンロードをサポート

#### オーディオ抽出
- ビデオからオーディオを抽出
- 多様なオーディオ形式 (MP3, M4A, AAC など) をサポート
- メタデータとサムネイルを自動的に埋め込み

#### 字幕ダウンロード
- 自動生成された字幕のダウンロード
- アップローダーが提供した字幕のダウンロード
- 複数の字幕形式をサポート
- ビデオへの字幕の埋め込みをサポート

### 3. 高度な機能

#### フォーマット選択
- 柔軟なフォーマットセレクター
- 条件付きフォーマット選択
- 複数のビデオ/オーディオストリームの結合をサポート

#### プロキシサポート
- HTTP/HTTPS/SOCKS プロキシをサポート
- プロキシチェーンをサポート

#### 認証
- ユーザー名/パスワードによるログインをサポート
- クッキーファイルをサポート
- OAuth をサポート

#### ダウンロード管理
- 中断されたダウンロードの再開
- 同時ダウンロード
- 速度制限
- ダウンロードキューの管理

## youtube-dl との違い

| 機能 | yt-dlp | youtube-dl |
|------|--------|------------|
| 更新頻度 | 頻繁に更新 | 更新が少ない |
| サイトサポート | 1000+ | 少ない |
| ダウンロード速度 | より速い | 比較的遅い |
| 機能特性 | 豊富 | 基本機能のみ |
| コミュニティの活発さ | 活発 | 比較的低い |

## 注意事項

### 法的コンプライアンス
- お住まいの地域の法律および規制を遵守してください
- コンテンツ制作者の著作権を尊重してください
- ダウンロードする権利があるコンテンツのみをダウンロードしてください
- 商業目的で使用しないでください（許可されている場合を除く）

### 使用上の推奨事項
- ダウンロード速度はネットワーク環境に左右される場合があります
- 一部のサイトでは、ダウンロードにログインや支払いが必要な場合があります
- 最高の体験を得るために、最新の yt-dlp バージョンを使用することをお勧めします
- 問題が発生した場合は、GitHub Issues を確認するか、バージョンを更新してください
